# LoveCount Site

Static site hosting LoveCount app's privacy policy and terms of service.

- **Privacy Policy**: [privacy.html](./privacy.html)
- **Terms of Service**: [terms.html](./terms.html)

## About LoveCount

LoveCount is an Android app that displays anniversary counts for couples via lock screen and home screen widgets.

## Updating content

When updating legal text, ensure the content here matches `core/ui/src/main/res/values/strings.xml` in the main LoveCount Android project. Both should be updated together to avoid inconsistency.
