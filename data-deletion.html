データ削除リクエスト

対象アプリ: LoveCount（パッケージ名: com.lovecount.app）
運営者: Pocket Bloom Studio
最終更新日: 2026年4月19日

LoveCount（以下「本アプリ」）では、ユーザーが自身に関連するデータの削除をいつでもリクエストできます。本ページでは、データの種類別に削除方法をご案内します。

1. アプリ内に保存されているデータ（即時削除可能）

以下のデータは本アプリの端末内にのみ保存されており、ユーザー自身で即座に削除できます。

データ種別保存場所削除方法
ニックネーム端末内アプリをアンインストール、または Android の「設定 → アプリ → LoveCount → ストレージ → データを削除」
記念日の日付・時刻端末内同上
写真端末内同上
アプリ設定（テーマ、通知設定等）端末内同上
アプリのアンインストールで完全削除されます。
本アプリはこれらのデータを外部サーバーには一切送信しません。端末から本アプリを削除すれば、関連するすべてのローカルデータは完全に消去されます。

2. 分析・クラッシュデータ（リクエストによる削除）

本アプリは Firebase（Google LLC 提供）を用いて、サービス改善のための匿名の利用統計情報およびクラッシュ情報を収集しています。

データ種別サービス保持期間（既定）
アプリ操作ログ（画面遷移等）Firebase Analytics最長14ヶ月
クラッシュ情報Firebase Crashlytics最長90日
端末・OS 情報（モデル、OS バージョン等）Firebase Analytics最長14ヶ月
広告識別子（Advertising ID）Firebase Analytics最長14ヶ月

これらのデータは個人を直接特定できない匿名情報ですが、削除をご希望の場合は以下のいずれかの方法でリクエストしてください。

方法 A: お問い合わせフォームから削除リクエスト

下記のお問い合わせフォームから、「データ削除リクエスト」と明記してご連絡ください。

お問い合わせの際は、以下の情報をご記載いただくとスムーズです（任意）:

  • 端末モデル（例: Google Pixel 7a）
  • Android バージョン
  • アプリ内でご使用のニックネーム（特定の補助として）
  • おおよその利用期間

お問い合わせフォームへ

リクエストの受領後、30日以内に Firebase から該当データの削除を実施し、ご連絡いたします。

方法 B: 広告識別子のリセット（自己対応）

Android 端末の「設定 → プライバシー → 広告 → 広告 ID を削除」から、ご自身で広告識別子をリセットできます。これにより、以降は別の匿名 ID として扱われます。
